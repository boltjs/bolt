Overview

Bolt is a module system for JavaScript.

It allows JavaScript applications and libraries to split their code into separate files to allow the reuse of components and encourage sustainable JavaScript development, whilst also providing compilation to a minimal set of files for production deployment and performance.

Bolt consists of a runtime module system, a compiler (jsc) for building optimised production distributables, a suite of tools to assist with managing projects (bolt) and a framework for writing componentized JavaScript.

Docs

This documentation is split into three main areas, each dealing with a different aspect of bolt development:

  • Use

    Documentation for the majority of bolt users. This explains bolt concepts, how to set up a new bolt project and how to write applications and libraries using bolt.

  • Extend

    Information on how to extend bolt to handle new kinds of resources that bolt doesn't handle out-of-the-box. Extensions to bolt are written using bolt itself, so an understanding of the topics covered in the Use section are a prerequisite.

  • Develop

    Design documents and architecture information for developers working on the bolt system itself.
